# Видео-поиск (учебный проект)

Учебный проект по вёрстке из курса **Яндекс Практикум**.  
Страница с видеоплеером и списком карточек видео, свёрстанная по макету.  
Логика работы уже реализована в готовом JavaScript-файле, я работала только с HTML и CSS.

## Что умеет страница

- Показывает список видео и текущее видео в большом плеере.
- По клику по карточке меняется выбранный ролик.
- Форма поиска фильтрует видео по параметрам.
- Есть прелоадеры при загрузке и блок ошибки при сбое.

## Технологии

- HTML5, шаблоны `<template>`
- CSS3, Flexbox, кастомные чекбоксы, состояния `:hover / :active / :focus-visible`
- Подключены шрифты Oswald и Fira Sans Condensed
